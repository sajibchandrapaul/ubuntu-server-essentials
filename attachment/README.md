### Practice Evidence / Screenshots

This section contains practical implementation evidence, including screenshots of configurations, commands, and system changes performed during the project. These screenshots demonstrate the real-world application of the concepts covered in this documentation.

The evidence may include:

* System configuration steps
* Command execution results
* Security hardening implementations
* Service status verification
* Configuration file modifications
* Testing and validation results

Each screenshot is included to provide a clear reference for the practical work completed during the project.
